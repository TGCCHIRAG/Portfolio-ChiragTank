News Digest

The Personalized News Digest app is a modern, AI-driven news aggregation platform designed to streamline news consumption by delivering curated and summarized content based on user preferences. The frontend is built using React and Tailwind CSS, ensuring a smooth and visually appealing user experience. For backend services, Nhost is utilized to manage authentication, database operations, and real-time updates efficiently. To provide intelligent news summarization and sentiment analysis, the app integrates n8n workflows with OpenRouter, enabling AI-powered processing of news articles. The application fetches real-time news using the GNews API and employs a filtration mechanism to remove duplicate or irrelevant articles, ensuring users receive only high-quality, non-redundant content.

One of the key features of this project is its automation pipeline, which continuously gathers, processes, and delivers news updates without requiring manual intervention. The system categorizes articles based on topics, relevance, and sentiment, allowing users to personalize their news feed according to their interests. Additionally, the platform is designed to be scalable and efficient, ensuring minimal latency in delivering fresh news updates. By combining API integrations, AI-powered processing, and real-time data management, the Personalized News Digest app offers a next-level news consumption experience, making it easier for users to stay informed without being overwhelmed by information overload.
